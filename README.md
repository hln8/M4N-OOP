# M4N-OOP

This repository contains the source code for the M4N-OOP project.

## Structure

- **server-api**: Java Spring Boot backend application.
- **desktop-app**: JavaFX desktop client application.
- **database**: Database schemas and migration scripts.
- **docs**: Project documentation.
